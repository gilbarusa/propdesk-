Regenerate Signed Lease PDF

Rebuilds the signed PDF for one countersigned lease from its body_html_snapshot + lease_signers, uploads a fresh copy to leases-signed, and updates leases.signed_pdf_url. Use this to fix leases whose existing signed PDF rendered blank because of the html2canvas timing bug. Does NOT touch tenant_pdf_bucket, tenant_pdf_path, tenant_documents, lease_document_assignments, or any P2 plumbing.

Preconditions
1. The lease row has status = 'countersigned', a landlord signer with signature_png, and a non-empty body_html_snapshot.
2. You are an admin — this tool talks to Supabase directly with the anon key under existing RLS policies.
3. This page pulls the same CONFIG.SUPABASE_URL / CONFIG.SUPABASE_KEY that the main app uses.
